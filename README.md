# Termo (Python MVP)

Um clone simples do jogo Termo/Wordle, focado em lógica pura e simplicidade.

## Filosofia do Projeto (MVP)

*   **Sem Frameworks:** Apenas Python padrão.
*   **Sem Classes Complexas:** Lógica procedural e estruturas de dados simples.
*   **Foco em DevOps:** Este projeto servirá de base para estudos de Docker e CI/CD.

## Como Rodar

1.  Crie o ambiente virtual: `python3 -m venv venv`
2.  Ative o ambiente: `source venv/bin/activate`
3.  Rode o jogo: `python termo.py`
